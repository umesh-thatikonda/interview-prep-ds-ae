## Test Overview
- **Time**: 90 minutes
- **Languages**: Python (stdlib + pandas) + SQL only

- **Goal**: Pass all tests as fast as possible — not clean code

## Level Expectations
| Level | Likely Focus |
|---|---|
| 1 | Data loading & cleaning (nulls, types, invalid rows) |
| 2 | Joining datasets (trips + stations) |
| 3 | SQL queries (aggregations, filters, joins) |

## Strategy
1. Copy the interface class methods into your file first
2. Read the docstring carefully — that's your spec
3. Run tests early and often (no penalty)
4. If description is ambiguous, let the tests tell you the truth
5. Hardcode edge cases if needed — just pass the test
